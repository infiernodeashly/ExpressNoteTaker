## Install

Open your integrated terminal and type npm i to install fs, path, and express.

## Usage

1. The user will open their integrated terminal and type "node server.js". 2. A message will be returned saying it is listening to the port specified. 3. The user will select the link and be led to a webpage with the URL. 4. The user will select "Get Started". 5. Select and complete the Note Title and Note Text fields, and select the icon of a floppy disk at the top-right to save the note. 6. Select the trashcan next to any notes to delete them. 7. Select any Note Title to view the Note Text. [![Watch the video] (https://youtu.be/jqArRpLA3mw) ![](./asset/ExpressNoteTaker.gif)
